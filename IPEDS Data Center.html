How provisional and final data are used by various Data Tools
Timing of IPEDS Provisional and Final Data Releases

Quality control procedures and institution follow-up continue two to three months after a collection period closes. Missing data are imputed due to nonresponse to entire survey components or items. The provisional release of the edited and imputed data will include a set of WEB tables that formerly appeared in IPEDS First Look reports as well as a Data Release Memo that summarizes response rate information for each survey. The final data release is available after institutions have had a chance to revise their data in the following year. Final data are not issued with a set of WEB tables and a Data Release Memo.

The timing of the releases can vary somewhat depending on the nature of the internal process, but generally will follow this timetable:

Collection Period Survey Components Collection closes Provisional data Final (Revised) data
FALL IC, C, E12 November Approximately 7 months after collection closes Approximately 7 months after institutions have revised their data the following year
WINTER ADM, GR, SFA, OM March
SPRING EF, F, HR, AL April
WEB Tables and Data Release Memo Yes No

The current review process for the data releases began with the 2018-19 IPEDS survey year and continues to undergo improvements. However, approval of the data release is subject to multiple steps in the review process and unanticipated delays can occur. The goal of NCES is to maintain or improve the timeliness of data release on an annual basis.

Provisional release data have undergone all NCES data quality control procedures. Data are imputed for nonresponding institutions. These data are used for First Look (Provisional Data) publications. The most recent data year available is listed for each survey.

Final release data include revisions to the provisional release data that have been made by institutions during the subsequent data collection year. The final release data can be used when the most up to date data are required; however, these data may not match tables from the First Look reports or web tables that were based on provisional data. Original data are the data that were reported to NCES during the initial data collection. Institutions may have updated these data in subsequent years; however, any changes made will not be reflected here, and as such, these data will match the published NCES tables. Revised data are data that have been updated by institutions during the subsequent data collection year. These data can be used when the most up to date data are required; however, they will not match the published NCES tables which are based on original data.
